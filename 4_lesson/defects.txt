Найденные проблемы в классе StringUtils:

1. В методе contains():
   - Проблема: используется сложный способ проверки через try-except
   - Лучше сделать проще: return symbol in string

2. В методе trim():
   - Проблема: удаляет только обычные пробелы
   - Нужно исправить: использовать string.lstrip() для удаления всех пробелов в начале

3. В методе delete_symbol():
   - Проблема: делает лишнюю проверку через contains()
   - Исправление: просто использовать string.replace(symbol, "")

4. Общая проблема:
   - Нет проверок на неправильный тип данных
   - Нужно добавить проверку типов в начале каждого метода
